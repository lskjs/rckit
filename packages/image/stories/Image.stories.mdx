# Lsk.js – Image

> Компонент **Image** используется для отображения изображения. Фаллбеки и разные условия

## Установка
```bash
npm i @lskjs/image
```


## Использование
```jsx
import { Image } from '@lskjs/image';

<Image src="https://avatars.mds.yandex.net/get-pdb/750514/5798b2df-9998-4381-b8be-57aaf7b65e92/s375" />
```
