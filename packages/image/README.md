## Installation and usage

The easiest way to use is to install it from npm and build it into your app with Webpack.

```bash
npm install @lskjs/image
```

Then use it in your app:

```jsx
import Image from '@lskjs/image';

const App = (
  <Image src="primary" />
);
```
